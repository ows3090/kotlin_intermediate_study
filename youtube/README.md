## MotionLayout
MotionLayout은 앱에서 모션과 위젯 애니메이션을 관리하는 데 사용할 수 있는 레이아웃으로 MotionLayout은 ConstraintLayout의 서브ㅡ클래스이며 제약 설정과 같은 다양한 레이아웃 기능들이 포함되어 있습니다. ConstraintLayout 라이브러리 일부인 MotionLayout은 추가 라이브러리 설정 없이 바로 사용 가능합니다.
